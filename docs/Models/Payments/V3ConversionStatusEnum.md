# V3ConversionStatusEnum

Lifecycle of a conversion.
`PENDING` — accepted by the PSP, not yet settled.
`COMPLETED` — settled, terminal.
`FAILED` — rejected or reverted, terminal. See `error`.



## Values

| Name        | Value       |
| ----------- | ----------- |
| `Unknown`   | UNKNOWN     |
| `Pending`   | PENDING     |
| `Completed` | COMPLETED   |
| `Failed`    | FAILED      |
